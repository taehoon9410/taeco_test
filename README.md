## git 명령어를 익혀봅시다.

- git clone : 저장소 이름 받아오기
- git status : 내용확인(동기화확인)
- git add 파일이름 : 파일첨부하기
- git commit -m "내용" : 필요내용 작성하기
- git push -u origin master : 최초의 전송하기
- git push : 최초이후 전송
- git config --global user.name "name" :  콤퓨타에 최초로 사용자 등록
- git config --global yser.email "mail"    :  콤퓨타에 최초로 사용자 메일등록
